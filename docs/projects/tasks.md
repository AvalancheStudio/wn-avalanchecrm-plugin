# Tasks

Efficiently manage daily work items within your projects using the Tasks module.

## Task Features

- **Assignment**: Assign tasks to specific staff members with clear deadlines.
- **Status Tracking**: Move tasks through customisable statuses (e.g., To Do, In Progress, Review, Done).
- **Time Tracking**: Log time spent on individual tasks for accurate billing and productivity analysis.
- **Prioritisation**: Set task priority levels to ensure the most important work is tackled first.
- **Comments & Mentions**: Discuss tasks directly with team members using integrated commenting.
