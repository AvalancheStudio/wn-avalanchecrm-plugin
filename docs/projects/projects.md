# Projects

The Projects module is designed to help you organise and track complex work from inception to completion.

## Project Management

- **Project Tracking**: Monitor the status, deadline, and progress of each project.
- **Team Collaboration**: Assign multiple staff members to a project and manage their permissions.
- **Milestones**: Break down large projects into manageable phases with specific goals.
- **Client Visibility**: Choose which project details and updates are visible to your clients in their portal.

## Kanban View

Visualise your project pipeline with an interactive Kanban board to easily see which stage each project is in.
