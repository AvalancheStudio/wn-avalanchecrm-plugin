# Support Tickets

The Support module provides a robust framework for handling customer inquiries and technical issues.

## Ticketing System

- **Multi-Channel Support**: Manage tickets raised via the client portal or internal staff entries.
- **Categorisation**: Organise tickets by type (e.g., Bug, Feature Request, Question) and department.
- **Status & Priority**: Set clear expectations with priority levels and custom status workflows.
- **Internal Notes**: Staff can communicate privately on tickets to resolve issues before replying to the client.
- **SLA Tracking**: Monitor response times to ensure quality service is maintained.
